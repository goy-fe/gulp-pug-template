# Gulp Pug Template

基于 __[Gulp](https://gulpjs.com/)__ 与 __[Pug](https://pugjs.org/api/getting-started.html)__ 的网站类项目模板。

## 集成功能

- SCSS
- Postcss
- Base64
- Babel
- Pug
- Imagemin
- Clean-CSS
- Autoprefixer
- Uglify
- Plop

支持配置定制，使用 `yaml` 来编写配置文件。

支持通过 __plop__ 简单自动化创建页面相关文件(HTML, SCSS, JS, Config)。

通过以下命令：

``` bash
$ yarn make
```
